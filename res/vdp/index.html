Introduction

"The goal of this doc is to provide beginers some informations about the MD VDP.
Why the VDP? Because this is the most important and the most virtual (ie:Hard to understand) part of the MD.
Why "pics"? Because it exists several docs about the VDP (Technical Overview, CMD's Doc, Programming faq 6, Devster's overview, Programming Intro...) and I wanted to make a new-style doc with no text. Of cource this doc isn't sufficent to understand how the VDP works but it's a cool preview.
It's free; Have fun!." Fonzie
